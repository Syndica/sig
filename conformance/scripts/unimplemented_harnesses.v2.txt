# relative to env/test-vectors/. Any fixture whose path equals an entry below
# or lives under one (matched at a directory boundary) is skipped.

# scripts/misc_failures.v2.txt is a separate, file-level CI denylist for
# individual `.fix` files that are known to fail against v2.
#
# See scripts/unimplemented_harnesses.v1.txt for the v1 counterpart.
